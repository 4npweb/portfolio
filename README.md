# Personal Portfolio

My personal portfolio and web development showcase, built with React and Vite.

**Live site:** [4np.dev](https://4np.dev)

---

## Stack

- React
- Vite

---

## Features

- Fully responsive
- Animated sections with Framer Motion
- Individual project pages with live and GitHub links
- Contact form via Web3Forms
- SEO optimized with Open Graph

---

## Pages

| Page | Description |
|---|---|
| `/` | Homepage — hero, about, work, services, contact |
| `/work` | Full project catalogue with filtering |
| `/work/project/:id` | Individual project page |

---

## Running locally

```bash
git clone https://github.com/4npdev/portfolio
cd portfolio
npm install
npm run dev
```
